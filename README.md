
# ♟️ AI Strategic Chess Coach
> **"스톡피시의 수치적 언어를 인간의 전략적 통찰로 번역합니다."** 

이 프로젝트는 체스 엔진의 평가값(+1.5, -0.8)은 보지만 그 이유를 이해하지 못해 정체기에 빠진 레이팅 1200 이상의 중급자들을 위해 제작되었습니다.  스톡피시의 강력한 계산 능력과 LLM의 추론 능력을 결합하여, 실시간으로 전략적 의도를 설명해주는 개인 맞춤형 AI 코칭 서비스입니다.

---

## 🛠️ 설치 및 실행 방법 (Quick Start)

### 1. 체스 엔진 설치 (`setup.py`)
이 프로젝트는 **Stockfish 18** 엔진을 사용하여 연산을 수행합니다. 아래 파일을 더블 클릭하여 실행하면 현재 폴더에 운영체제에 맞는 엔진 파일이 자동으로 다운로드 및 설치됩니다.

```bash
setup.py
```

### 2. 필수 라이브러리 설치
프로그램 실행을 위해 UI 라이브러리와 체스 로직 관련 라이브러리를 설치해야 합니다.

```bash
pip install PyQt6 python-chess requests
```

### 3. 프로그램 실행
로컬 LLM 서버(LM Studio)가 실행 중인지 확인한 후, 다음 파일을 더블 클릭하여 실행하세요. `example_game.pgn` 파일을 통해 실제 기보 분석 기능을 바로 체험해볼 수 있습니다.

```bash
main.py
```

## 🧠 시스템 아키텍처 및 요구 사항

### 1. 로컬 LLM 서버 연동 (LM Studio)
이 서비스는 보안과 성능을 위해 로컬 환경의 LLM을 사용합니다.
* **서버 주소:** `http://127.0.0.1:1234`
* **필요 모델:** `google/gemma-3-12b` (LM Studio에서 Load 및 Start Server 필수, 32G RAM 필요)


### 2. 주요 기능 
* **Real-time Analysis:** 스톡피시가 3초간 깊게 수 읽기를 수행한 후 최적의 라인을 도출합니다. 
* **Strategic Interpretation:** LLM이 수치의 이면(중앙 점유, 기물 활동성 등)을 인간의 언어로 설명합니다. 
* **Interactive UI:** 기보 리스트 클릭 및 화살표 키를 통해 체스닷컴과 유사한 분석 경험을 제공합니다.

## 🎯 프로젝트 기대 효과 
* **직관적 학습:** 단순 수순 암기가 아닌 논리적 원리 이해를 통한 실력 향상.
* **정체기 해소:** 엔진만으로는 알 수 없던 자신의 나쁜 습관을 통계적/심리적으로 교정.
* **교육의 민주화:** 고가의 1:1 코칭 없이도 수준 높은 개인 맞춤형 피드백 확보.
